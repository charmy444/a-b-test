# A/B test

Простой анализ результатов A/B-теста.

В данных есть:

- `user_id`
- `group`
- `views`
- `clicks`

Основная метрика: `CTR = clicks / views`.

## Что сделано

В ноутбуке `Untitled.ipynb`:

- загружены данные
- проверены пропуски и дубликаты
- построены простые графики
- посчитана CTR для test и control
- проведен тест пропорций
- дополнительно проведен тест Манна-Уитни

## Результат

В тестовой группе CTR вырос:

- средняя пользовательская CTR: примерно на 11.52%
- агрегированная CTR: примерно на 11.05%

p-value получилось маленьким, поэтому различие можно считать статистически значимым.
